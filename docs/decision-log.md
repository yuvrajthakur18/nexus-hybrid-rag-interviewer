# Key Architectural Decisions

1. **FastAPI** for async APIs and production-ready Python ecosystem.
2. **PostgreSQL** for relational interview/domain data.
3. **Qdrant** for fast vector retrieval over PDF-derived chunks.
4. **Local face embeddings** to satisfy identity verification without external vendor lock-in.
5. **Parallel dual-answer agents + evaluator** to improve answer quality in hybrid questions.
